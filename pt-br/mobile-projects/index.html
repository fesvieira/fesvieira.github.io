Champions Round

No aplicativo Champions Round, adquiri valiosa experiência profissional trabalhando como parte das equipes Android e iOS. Colaborando com essas equipes, tive a oportunidade de contribuir para o desenvolvimento de vários modos de jogo e funcionalidades de chat, incorporando recursos avançados como Web Sockets e tratamento de solicitações de API Rest.

O aplicativo Champions Round oferece uma experiência emocionante de fantasy sports com vários modos de jogo para entusiastas de esportes. Notavelmente, "Spicy Slips" é um jogo de ação ao vivo onde os jogadores constroem jogadores e props de equipe em tempo real durante transmissões de jogos, com a chance de ganhar fazendo escolhas corretas. Além disso, "Daily Dozen" é um jogo de escolha que oferece uma seleção diária de doze props para prever, e "Boost Ball" multiplica estrategicamente a pontuação de um atleta por 2x a cada semana. Outros modos como "Seven Stakes", "Sunday's Best" e "Big Board Showdown" oferecem desafios únicos para fãs de esportes que desejam competir e se divertir online com amigos.

Este aplicativo oferece uma experiência diversificada e interativa para entusiastas de fantasy sports, com modos de jogo envolventes e o benefício adicional de conversar com amigos, permitindo que os usuários se conectem e compitam em desafios emocionantes. É uma escolha ideal para aqueles que buscam uma experiência esportiva única e social.

Whatever Weather

WhateverWeather é um aplicativo móvel dinâmico e intuitivo projetado para fornecer aos usuários previsões meteorológicas atualizadas para uma localização especificada. Aproveitando tecnologias de ponta como Retrofit, Datastore, Dagger Hilt e Jetpack Compose, o WhateverWeather oferece uma experiência perfeita e imersiva para se manter informado sobre as condições climáticas.

Recursos Busca dados de previsão do tempo em tempo real do WeatherApi usando Retrofit. Armazena preferências do usuário e dados em cache usando Datastore. Utiliza Dagger Hilt para injeção de dependência, aprimorando a modularidade e testabilidade. Desenvolve uma interface de usuário visualmente envolvente com Jetpack Compose.

Hábitos & Metas

O HabitGoalsApp é um rastreador de hábitos de ponta projetado com 100% de componentes de IU Jetpack Compose, garantindo uma experiência de usuário fluida e visualmente envolvente. Com Room integrado para SQL e gerenciamento de banco de dados, os usuários podem armazenar e gerenciar facilmente seus dados de hábitos, garantindo integridade e recuperação de dados. Além disso, o aplicativo é aprimorado com gestos intuitivos de deslizar, tornando-o fácil de usar e conveniente para rastrear hábitos e definir metas, tudo dentro de uma interface de usuário elegante e moderna.

PomoTasks

PomoTasks é uma fusão do Cronômetro Pomodoro e uma lista de tarefas, permitindo que você organize sua rotina em blocos de tempo. É um aplicativo de tela única com um relógio central que permite definir e controlar seu temporizador (pausar e reproduzir). Abaixo do relógio central, há uma lista de tarefas onde você pode marcar tarefas como concluídas, excluir, modificar e removê-las permanentemente do armazenamento.

Champions Round

Within the Champions Round app, I gained valuable professional experience working as part of both the Android and iOS teams. Collaborating with these teams, I had the opportunity to contribute to the development of numerous game modes and chat functionalities, incorporating advanced features such as Web Sockets and handling Rest API Requests.

The Champions Round app offers an exciting fantasy sports experience with various game modes for sports enthusiasts. Notably, "Spicy Slips" is a live-action game where players build player and team props in real-time during game broadcasts, with the chance to win by making correct choices. Additionally, "Daily Dozen" is a pick'em-style game providing a daily selection of twelve props to predict, and "Boost Ball" strategically multiplies an athlete's score by 2x each week. Other modes like "Seven Stakes," "Sunday's Best," and "Big Board Showdown" offer unique challenges for sports fans looking to compete and have fun online with friends.

This app offers a diverse and interactive experience for fantasy sports enthusiasts, with engaging game modes and the added benefit of chatting with friends, allowing users to connect and compete in exciting challenges. It's an ideal choice for those seeking a unique and social sports experience.

Whatever Weather

WhateverWeather is a dynamic and intuitive mobile application designed to provide users with up-to-date weather forecasts for a specified location. Leveraging cutting-edge technologies such as Retrofit, Datastore, Dagger Hilt, and Jetpack Compose, WhateverWeather delivers a seamless and immersive experience for staying informed about weather conditions.

Features Fetches real-time weather forecast data from WeatherApi using Retrofit. Stores user preferences and cached data using Datastore. Utilizes Dagger Hilt for dependency injection, enhancing modularity and testability. Develops a visually engaging user interface with Jetpack Compose

Habits & Goals

The HabitGoalsApp is a cutting-edge habit tracker designed with a 100% Jetpack Compose UI component, ensuring a seamless and visually engaging user experience. With integrated Room for SQL and database management, users can effortlessly store and manage their habit data, guaranteeing data integrity and retrieval. Furthermore, the app is enhanced with intuitive swipe gestures, making it user-friendly and convenient for tracking habits and setting goals, all within an elegant and modern user interface.

PomoTasks

PomoTasks is a fusion of the Pomodoro Timer and a task list, allowing you to organize your routine in blocks of time. It is a single-screen app featuring a central clock that enables you to set and control your timer (pause and play). Below the central clock, there is a task list where you can mark tasks as completed, delete, modify, and permanently remove them from storage.
